📅 第 1 天：巴金森病的認識與成因

巴金森病的認識與成因

巴金森病是一種神經退化性疾病，於1817年被發現，至今確切原因不明，主要病理變化是中腦黑質部製造多巴胺的神經細胞退化凋亡，導致大腦無法正常傳遞訊號，影響動作控制。

  1. 影響多巴胺產生，導致動作障礙。
  2. 好發於50歲以上中老年人，但年輕人也可能罹患。
  3. 疾病進展緩慢，不可逆但可透過治療控制。
  4. 可能與遺傳、環境毒素或頭部外傷有關。

真實世界範例：著名拳擊手穆罕默德·阿里長期罹患巴金森病，影響其動作與言語，但透過治療維持生活品質。

實踐行動計劃：

  1. 了解家族病史，評估個人風險。
  2. 避免暴露於農藥或重金屬等環境毒素。
  3. 維持健康生活習慣，如均衡飲食與規律運動。
  4. 定期進行健康檢查，及早注意症狀。

📅 第 2 天：症狀與診斷

症狀與診斷

巴金森病的診斷主要依賴臨床症狀觀察，無單一檢驗可確認，醫師透過病史、體檢及反應藥物測試診斷，早期發現有助控制病情。

  1. 典型症狀包括顫抖、僵硬與動作緩慢。
  2. 診斷需排除其他類似疾病，如本質性顫抖。
  3. 影像學如MRI可輔助排除其他病因。
  4. 疾病分期幫助評估進展與治療。

真實世界範例：一位60歲男性出現手抖與步態不穩，經醫師檢查與左旋多巴測試確認為巴金森病。

實踐行動計劃：

  1. 注意早期症狀，如嗅覺減退或便秘。
  2. 尋求神經專科醫師評估。
  3. 記錄症狀變化以供醫師參考。
  4. 接受必要影像檢查。

📅 第 3 天：動作症狀

動作症狀

巴金森病的動作症狀是主要特徵，包括靜止性顫抖、肌肉僵硬、動作緩慢與姿勢不穩，影響日常生活。

  1. 靜止性顫抖：休息時手或腳抖動。
  2. 肌肉僵硬：肢體抵抗被動運動。
  3. 動作緩慢：日常動作變得遲緩。
  4. 姿勢不穩：易跌倒，平衡問題。

真實世界範例：患者寫字時字跡變小（微字症），影響工作效率。

實踐行動計劃：

  1. 練習大動作運動，如太極拳。
  2. 使用輔助工具，如手杖。
  3. 定期復健治療。
  4. 避免滑溜地面以防跌倒。

📅 第 4 天：非動作症狀

非動作症狀

除了動作問題，巴金森病常伴隨非動作症狀，如睡眠障礙、憂鬱、認知變化，需全方位照護。

  1. 睡眠問題：快速眼動睡眠障礙。
  2. 情緒變化：憂鬱或焦慮。
  3. 自主神經症狀：便秘或低血壓。
  4. 認知障礙：記憶力減退。

真實世界範例：患者出現便秘與嗅覺喪失，早在動作症狀前10年出現。

實踐行動計劃：

  1. 維持規律作息改善睡眠。
  2. 尋求心理諮詢緩解情緒問題。
  3. 飲食調整預防便秘。
  4. 腦力訓練維持認知。

📅 第 5 天：藥物治療

藥物治療

藥物是巴金森病主要治療方式，補充多巴胺或模擬其作用，緩解症狀，但需注意副作用。

  1. 左旋多巴：主要藥物，轉化為多巴胺。
  2. 多巴胺促效劑：模擬多巴胺作用。
  3. MAO-B抑制劑：延長多巴胺效用。
  4. 監測副作用如異動症。

真實世界範例：患者使用左旋多巴後動作改善，但長期出現藥效波動。

實踐行動計劃：

  1. 依醫囑準時服藥。
  2. 記錄藥效與副作用。
  3. 避免自行調整劑量。
  4. 定期回診調整治療。

📅 第 6 天：手術與先進治療 / 生活方式管理：運動與飲食

手術與先進治療

當藥物失效時，手術如深腦刺激術（DBS）可改善症狀，適用於中晚期患者。

  1. DBS：植入電極刺激腦部。
  2. 聚焦超音波：非侵入性治療顫抖。
  3. 評估適合手術條件。
  4. 術後調整與追蹤。

真實世界範例：一位患者接受DBS後，顫抖明顯減少，提升生活品質。

實踐行動計劃：

  1. 諮詢醫師評估手術適合性。
  2. 了解手術風險與益處。
  3. 術後遵守復健計劃。
  4. 定期檢查裝置功能。

生活方式管理：運動與飲食

規律運動與均衡飲食可延緩巴金森病進展，維持身體機能與營養。

  1. 有氧運動：如散步或游泳。
  2. 力量訓練：改善肌肉僵硬。
  3. 地中海飲食：富含蔬果與 omega-3。
  4. 補充維生素D與抗氧化物。

真實世界範例：患者透過每日瑜伽練習，改善平衡並減少跌倒風險。

實踐行動計劃：

  1. 每週150分鐘中等強度運動。
  2. 攝取高纖飲食預防便秘。
  3. 避免咖啡因影響藥物吸收。
  4. 與營養師討論個人計劃。

📅 第 7 天：心理支持與家庭照護

心理支持與家庭照護

巴金森病影響心理健康，家庭照護者需提供情感支持，共同面對疾病。

  1. 處理憂鬱與焦慮。
  2. 支持團體參與。
  3. 照護者自我照顧。
  4. 規劃長期照護。

真實世界範例：家庭成員參與支持團體，幫助患者維持積極心態。

實踐行動計劃：

  1. 加入病友協會。
  2. 練習冥想減壓。
  3. 分配照護責任避免 burnout。
  4. 尋求專業心理諮詢。
