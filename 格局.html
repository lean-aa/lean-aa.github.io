第1天 🌅 人生的格局

人生的格局

本章探討如何提升個人格局，首先需認識到超出個人能力之外的力量，如道德或自然規律，並強調主動性、分享利益以及寬容他人之短。

  1. 上帝只垂青主動的人
  2. 分享利益，獨立決定
  3. 不要因為小惡而忘記大善
  4. 勿因人之短護己之短，勿以人之短炫己之長

真實世界範例

比爾·蓋茨在微軟成功後，分享財富成立比爾及梅琳達·蓋茨基金會，專注全球健康與教育，擴大其格局與影響力。

實踐行動計劃

  1. 每天主動完成一件超出常規的事務。
  2. 在團隊中主動分享資源或利益。
  3. 反思個人短處，避免以他人之短掩蓋己短。
  4. 培養敬畏之心，閱讀相關哲學書籍。

命運的力量

本章討論命運的影響力，強調盡人事聽天命、認識能力邊界，並學習如何吸引好運及止損。

  1. 盡人事，仍需聽天命
  2. 看清能力圈的邊界
  3. 怎樣做事才能獲得好運氣
  4. 捨得止損，才能斬斷厄運

真實世界範例

史蒂夫·喬布斯在被蘋果開除後，創立NeXT公司，最終回歸蘋果並帶領其復興，展現止損與轉向的能力。

實踐行動計劃

  1. 評估個人能力圈，避免超出邊界投資。
  2. 每日記錄三件積極行為以吸引好運。
  3. 設定止損點，在失敗時及時調整。
  4. 培養敬畏命運，閱讀相關書籍。

生活的節奏

本章強調生活需有節奏，避免成為積極廢人，學習休息與幸福之道。

  1. 要思考，就需要慢下來

  3. 懂得休息，才能更好生活
  4. 幸福的藍色地帶

真實世界範例

  3. 設定休息時間，培養幸福習慣。
  4. 探索藍色地帶生活方式，如地中海飲食。
