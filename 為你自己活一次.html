📅 第 1 天：明白原則

明白原則

明白是指不再人云亦云，不讓自己的心被他人操控，追求自我覺醒與獨立思考。

  1. 認清自己的真正需求。
  2. 避免盲從他人意見。
  3. 培養獨立思考能力。
  4. 透過抬槓自己尋求真諦。

真實世界範例

一位上班族總是跟隨同事的職業選擇，但後來發現不適合自己，轉而追求內心熱愛的藝術領域，生活變得更充實。

實踐行動計劃

  1. 每天花10分鐘反思一天的決定是否出於自己。
  2. 閱讀不同觀點的文章，訓練獨立判斷。
  3. 記錄情緒日誌，了解自己的反應。
  4. 與朋友討論時，勇敢表達自己的看法。

📅 第 2 天：認識自己

認識自己

深入了解自己的內在世界，認識優缺點及情緒來源，以建立堅實的自我基礎。

  1. 重視自我覺察。
  2. 識別情緒模式。
  3. 接受自己的不完美。
  4. 探索內心需求。

真實世界範例

一位名人面對公眾壓力，透過自我認識找到情緒平衡，避免燒盡。

實踐行動計劃

  1. 每天練習5分鐘冥想，觀察思緒。
  2. 寫自傳式日記，回顧人生經歷。
  3. 向信任的朋友尋求反饋。
  4. 設定小目標，測試自我界限。

📅 第 3 天：面對負面情緒

面對負面情緒

勇敢正視負面情緒，而不是逃避，讓它成為成長的動力。

  1. 承認負面情緒的存在。
  2. 分析情緒背後的原因。
  3. 轉化負面為正面能量。

真實世界範例

一位演員在失戀後，正視悲傷情緒，轉而創作音樂，獲得療癒。

實踐行動計劃

  1. 當負面情緒來臨時，深呼吸並命名它。
  2. 寫下情緒來源並 brainstorm 解決方案。
  3. 從事運動或藝術釋放情緒。
  4. 分享給親友，獲得支持。

📅 第 4 天：控制情緒

控制情緒

學習管理情緒，避免情緒主導行為，達成內在平衡。

  1. 辨識情緒觸發點。
  2. 練習延遲反應。
  3. 使用正念技巧。
  4. 培養情緒彈性。

真實世界範例

一位經理在會議中控制怒氣，轉而理性溝通，解決團隊問題。

實踐行動計劃

  1. 學習深呼吸技巧，每天練習。
  2. 設定情緒冷卻時間，避免衝動。
  3. 閱讀情緒管理書籍。

📅 第 5 天：接受自卑

接受自卑

接受自卑感，不讓它阻礙前進，而是轉化為自我提升的動力。

  1. 檢討自卑標準。
  2. 肯定自身價值。
  3. 避免比較他人。
  4. 尋求成長機會。

真實世界範例

一位學生接受學業自卑，專注於興趣領域，成為專家。

實踐行動計劃

  1. 列出個人優點清單。
  2. 挑戰自卑信念，找反例。
  3. 設定可達成目標，累積自信。
  4. 參與支持團體。

📅 第 6 天：放下評判他人 與 與自己對話

放下評判他人

停止對他人的評判，將心力投入自我成長，避免浪費能量。

  1. 認識評判的害處。
  2. 練習同理心。
  3. 專注內在投資。

真實世界範例

一位網民停止網路評判，轉而學習新技能，提升生活品質。

實踐行動計劃

  1. 察覺評判想法時，轉移注意力。
  2. 嘗試理解他人視角。
  3. 每天分配時間自我反思。
  4. 減少社群媒體使用。

與自己對話

定期與自己抬槓對話，澄清內心想法，達成真正明白。

  1. 靜下心來反思。
  2. 質疑既有信念。
  3. 探索內心衝突。
  4. 記錄對話過程。

真實世界範例

一位作家透過內心對話，解決創作瓶頸。

實踐行動計劃

  1. 找安靜地方，獨處10分鐘。
  2. 問自己問題，如「我真正想要什麼？」
  3. 寫下對話內容。

📅 第 7 天：定義自己的幸福

定義自己的幸福

不依賴他人標準，自行定義幸福，活出獨一無二的人生。

  1. 反思幸福來源。
  2. 設定個人目標。
  3. 追求內在滿足。
  4. 調整生活優先順序。

真實世界範例

一位企業家放棄高薪，追求家庭生活，找到真正幸福。

實踐行動計劃

  1. 列出使你快樂的事物。
  2. 排除社會期望的影響。
  3. 逐步實踐幸福計劃。
  4. 定期評估並調整。
